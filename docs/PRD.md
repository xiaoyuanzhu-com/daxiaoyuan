# 大大校园 PRD

## 名字

**大大校园** —「大」取大同、大公、有容乃大之意，主张大学应向所有人敞开。叠字「大大」是中文产品命名里成熟的萌化路径（哔哩哔哩、拼多多、美图秀秀），让一个本来厚重的价值主张以亲切的方式落地。

「大学」二字又典出儒家经典《大学》——这门"大人之学、君子之学"，本就不该是关上门做的。详见下方「《大学》参照」一节。

延展可能：「大大」可以是产品的吉祥物——胖嘟嘟、永远笑眯眯地敞着门的拟人形象。

## 背景

新冠之后，国内大量高校对社会公众关闭。市民失去了在校园里散步遛弯、使用图书馆、操场、体育馆等公共资源的机会。围绕「大学是否应当向公众开放」的讨论持续多年，但缺少一个集中、可查、可比较的信息入口：

- 哪些大学完全开放？
- 哪些只开放散步、不开放设施？
- 哪些需要预约 / 凭身份证 / 凭校友卡？
- 周末和工作日是否不同？
- 暑期 / 寒假是否不同？

普通人遇到具体问题（"我想周末带孩子去 XX 大学操场跑步，能进吗？"）时，只能靠搜索碎片化的小红书帖子和过期新闻。

## 价值观

1. **开放** — 大学是公共资源，应当向社会敞开
2. **人人** — 不分校友、不分身份、人人可入
3. **透明** — 把现状摊开来给所有人看，是推动改变的第一步

## 《大学》参照

「大大校园」名中之"大"，取大同、大公、有容乃大；而「大学」二字本身，又典出儒家经典《大学》。今日所谓"大学"，与《大学》所言的"大学"，本应是同一回事——成人之学、君子之学、为天下立心的学问。这门学问，从来不是关上门做的。

下面是《大学》中与「大大校园」主张直接相关的几段，供日后宣传、关于页、海报文案择用。

### 大学之道，在明明德，在亲民，在止于至善

> 大学之道，在明明德，在亲民，在止于至善。

（朱熹改本作"新民"，王阳明力主"亲民"。两义可并存。）

- **明明德** —— 彰显光明的德性
- **亲民 / 新民** —— 亲近百姓、使百姓维新
- **止于至善** —— 安立于至善的境界

大学之道首在"亲民"。一所将围墙关上、把市民隔绝在外的大学，与"亲民"的本义背道而驰。**「大大校园」想做的，是把"大学之道"从课本里的注释还原成街边生活的体验：让大学重新亲近它所在的城市、它所属的人民。**

### 知止而后有定

> 知止而后有定，定而后能静，静而后能安，安而后能虑，虑而后能得。

知道在哪里停下，心才能安定；安定之后，方能静、能安、能思、能得。

**"止"既是终点，也是栖身之所。** 大学的湖边、操场、图书馆，本该是市民可以"止"于其中的公共空间——一处可以停下来发呆、读书、跑步的地方。

### 苟日新，日日新，又日新

> 汤之盘铭曰："苟日新，日日新，又日新。"《康诰》曰："作新民。"《诗》曰："周虽旧邦，其命维新。"是故君子无所不用其极。

商汤刻在浴盆上的箴言：每一天都要更新自己。

封闭的校园是凝固的，开放的校园是流动的。**「大大校园」相信，大学只有在不断与社会发生交换、被不断打开和重塑的过程中，才能保持鲜活。项目本身的数据，也按"日日新"的节奏由用户共同维护——每一次"我今天去过"的反馈，都是一次小小的"作新民"。**

### 邦畿千里，惟民所止

> 《诗》云："邦畿千里，惟民所止。"

国家广袤千里的疆土，是百姓得以驻足栖居之地。

大学的园子、操场、图书馆、湖边长椅，也理应是市民可以"止"的地方——可以停留、可以休憩、可以坐下来读一会儿书。**当一座城市的公共空间被一道道围墙隔断，"惟民所止"就成了空话。**

### 物格而后知至

> 物格而后知至，知至而后意诚，意诚而后心正，心正而后身脩，身脩而后家齐，家齐而后国治，国治而后天下平。

要"知"，必先"格物"——亲身接触事物。

大学是知识生产之地；若知识只在墙内流通而不向社会敞开，便有"格物"而无"作新民"。**「大大校园」逐校逐门核对开放状况，本身也是一次"格物"——把弥散在搜索结果与小红书里的碎片，对齐成可被查验的事实。**

### 自天子以至于庶人，壹是皆以修身为本

> 自天子以至于庶人，壹是皆以修身为本。

不分贵贱身份，从天子到平民，一切都以"修身"为根本。

这一句对应「大大校园」的"人人"价值观：**校园是否对你开放，不应取决于你是不是校友、是不是教职工、是不是某种身份。修身是每个人都要走的路，因此每个人都有理由走进大学这个"修身之所"。**

### 君子贤其贤而亲其亲

> 君子贤其贤而亲其亲，小人乐其乐而利其利。

君子敬重贤者、亲近亲族；小人只为自己的快乐与利益打算。

一所大学若只服务于"自己人"（在校师生、校友），其余概不相干，便落入"乐其乐而利其利"的小人之境。**真正的大学，既以学术之德泽被同道，亦以公共之亲意泽被四邻。**

### 德者本也，财者末也

> 德者本也，财者末也。
> 国不以利为利，以义为利也。

机构之根本在德性，不在财货；治理之善在于义，不在于利。

大学坐拥土地、建筑、藏书、设施，这些是社会用税收与情感共同滋养出来的公共资产。**当大学把这些资源视为"本"（应当回馈社会的公器）而非"末"（须严防外人染指的私产），它就回到了《大学》所定义的"大学"本位。**

### 民之所好好之，民之所恶恶之

> 《诗》云："乐只君子，民之父母。"民之所好好之，民之所恶恶之，此之谓民之父母。

百姓喜欢的，便也喜欢；百姓厌恶的，便也厌恶——这才配称作"民之父母"。

**市民想进大学散步、晨跑、带孩子看银杏，这是"民之所好"。把它顺手满足，是低成本的"民之父母"姿态；把它拒之门外，则是把自己置于民意的对立面。**

---

附：「大同」典出《礼记·礼运》——与《大学》同属《礼记》系列，互为映照：

> 大道之行也，天下为公。⋯⋯故人不独亲其亲，不独子其子⋯⋯货恶其弃于地也，不必藏于己。⋯⋯是故谋闭而不兴，盗窃乱贼而不作，故外户而不闭，是谓大同。

**"故外户而不闭"——大同世界的标志之一，就是外面的门不必关。关上校门容易，打开却需要勇气与制度。**

## 形态

**首要：微信小程序**。轻量、即用、易传播，贴近真实使用场景（"在大学门口想知道能不能进"）。

**配套：H5 网页版**。便于在小程序之外的渠道分享（小红书 / 微博 / 浏览器直达），承担 SEO 入口与移动浏览器兼容。

**远期：iOS / Android 原生客户端**。需求成熟后再做。

各端各自用最合适的原生技术栈实现（Web 用 Vite + React，微信小程序用原生框架），通过中心后端提供的统一 HTTP API 共享同一份数据。

## 核心功能（MVP）

### 1. 学校开放状态查询
- 列表 / 地图两种入口
- 每所学校一张卡片，展示：
  - 开放等级（完全开放 / 限时开放 / 预约开放 / 仅校友 / 关闭）
  - 设施开放情况（散步 / 图书馆 / 操场 / 体育馆 / 食堂）
  - 进入方式（刷身份证 / 预约 / 校友卡 / 自由通行）
  - 时间段（工作日 / 周末 / 寒暑假差异）
  - 最后更新时间

### 2. 用户共建
- 用户可补充 / 纠正信息（"我今天去过，实际是这样"）
- 简单的可信度机制：最近多人确认 > 旧信息

### 3. 倡导内容
- 一个固定页面解释「为什么大学应该开放」
- 已开放学校的故事、用户的真实使用画面

## 非目标（MVP 不做）

- 不做预约系统（学校自有渠道）
- 不做社交 / 评论 / 点赞
- 不做账号体系（共建用最轻方式：手机号 or 微信授权）
- 不覆盖中小学

## 信息架构

```
首页（地图 + 我附近的学校）
├── 学校详情页
│   ├── 开放状态
│   ├── 设施清单
│   ├── 进入方式
│   └── 用户补充记录（时间倒序）
├── 城市列表页
└── 关于大大校园（价值主张）
```

## 数据来源与存储

**来源**
- **种子数据**：人工整理一线 + 新一线城市的 985/211 学校开放状态
- **持续更新**：用户共建为主
- **辅助来源**：抓取学校官网公告、小红书等公开内容做交叉验证

**存储**
- 学校信息 committed 在仓库的 `data/schools/<country>/<slug>.json`，启动时由后端
  全量 load 进内存。后端（Go + Gin，部署于 `ddxy.xiaoyuanzhu.com`）通过 HTTP API
  提供读写：写操作同时更新内存 map 并 atomic 落盘到对应 JSON。数据本身就是 git
  跟踪的，PRD §透明 的承诺直接由仓库可见性实现。
- Web、微信小程序、未来的 iOS/Android 客户端共用同一套 HTTP API

**透明**
- 全部 `GET` 接口公开免鉴权，任何人可直接调用——这是"透明"价值观在技术层面的兑现
- 提供全量数据导出端点（如 `/api/dump.json`），便于研究者、记者、社区做独立分析
- 计划：每日 GitHub Action 拉一份数据库快照 push 到公开仓库存档，留下可追溯的历史

## 命名演化记录

**第一轮**：项目主名发散
- 候选方向——大学公园 / 大学之大 / 大同校园 / 大校园
- 「**大校园**」中选：用「大」承载大同、大公、有容乃大的传统价值底色

**第二轮**：项目名 vs 小程序名 是否分开？
- 决定**统一**——同一个名字承担所有传播场景（项目内部、小程序商店、媒体传播），搜索 / 分享零损耗

**第三轮**：「大校园」在微信被注册（未上线），需要变奏

**Plan B**：如果「大大校园」最终也注册不下来，回退到 **大校园逛逛**（精神最接近，注册概率高）
